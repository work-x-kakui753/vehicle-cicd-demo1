# AGENTS.md

## Repository Purpose

- This repository is a vehicle-oriented C language CI/CD demo.
- Do not add confidential data, customer names, or real project identifiers.

## Tooling Rules

- Do not use paid external tools.
- Prefer configurations that run on Windows GitHub Actions.
- `scripts\ci_windows.bat` is the center of the CI flow.

## Change Rules

- After modifications, verify locally as much as the current environment allows.
- Keep `README.md` and files under `docs/` aligned with code and workflow changes.
- Preserve the artifact layout under `build/` so the demo remains easy to explain.
